⛳ 골프 라운딩 관리 앱

필드에서 바로 기록하고, 데이터로 내 골프를 분석하는 완전 오프라인 모바일 앱


📱 어떤 앱인가요?
서버도 없고, 로그인도 없고, 광고도 없습니다.
라운드가 끝난 뒤 적어두는 수첩처럼, 이 앱은 내 폰 안에 데이터를 저장합니다.
18홀 입력 → 자동 스코어 계산 → 통계 분석까지, 골프 기록에 필요한 것만 담았습니다.
Android APK로 설치하거나, 브라우저에서 바로 PWA로 사용할 수 있습니다.

✨ 주요 기능
🏠 홈 대시보드

KPI 4카드: 올해 평균 · 최근 10회 평균 · 목표 달성률 · 총 라운드 수
스코어 추이 차트: 최근 20라운드 흐름을 미니 SVG 차트로 시각화
최근 라운드: 5개 바로가기 (탭 → 상세 편집)

✏️ 라운드 입력 (2단계)
1단계 — 기본정보

날짜 / 부(1부·2부·3부) / 시간 / 날씨 / 기온
CC명 · 전반 코스 · 후반 코스 (초성 자동완성)
동반자 최대 3명 (이름 + 스코어 + 비고)
비용: 그린피 · 카트 · 캐디 · 기타 · 이벤트 (합계 자동 계산)
라운드 총평 메모

2단계 — 홀별 현장입력

PAR (3·4·5) 세그먼트
티샷 결과: G(그린) / E(엣지) / F(페어웨이) / R(러프) / B(벙커) / X(아웃)
방향: L / C / R / B
구역: P(페널티) / O(OB)
버디온 수 · 퍼트 수 → 스코어 자동 계산
멀리건 · 로스트볼
스코어 직접 조정 (자동값 재정의 가능)
홀 점프 도트 (전반 9홀 + 후반 9홀)
실시간 요약: 스코어 · 전/후반 · 퍼트 · 이글·버디·파·보기·더블·트리플

📋 라운드 목록

전체 목록 / 초성 검색
탭 확장 → 홀별 상세, 수정, 삭제
동반자 탭: 함께 친 사람별 통계

📊 통계 분석
기간 필터(올해 / 최근 3개월 / 전체 / 직접 설정) + 6개 탭
탭내용종합스코어·티샷·퍼트·로스트볼·버디온 평균, 등급 배지동반자함께 라운드한 사람별 내 평균 vs 상대 스코어분포이글·버디·파·보기·더블·트리플 분포, 홀별 약점비용총액 / 1회 평균, 항목별 비율목표연도 목표 vs 실제 달성 현황 (✓/✕)기준잘함·보통·아쉬움·노력 기준값 확인 및 편집
📈 리포트

월별 / 연도별 평균 집계표 (스코어 · 티샷아웃 · 퍼트 · 로스트볼)

🔒 보안

4자리 PIN 잠금 (SHA-256 해시 저장)
PIN 변경 · 해제 가능

💾 데이터 관리

.db 파일로 내보내기 (SQLite 바이너리)
다른 기기에서 가져오기 → 중복 없이 병합 (날짜+CC+부+시간 기준)
PC ↔ 폰 양방향 동기화 지원


🎨 등급 시스템
모든 지표에 4단계 색상 등급 적용:
등급색상스코어 기준잘함 🟢형광 초록79타 이하보통 ⬜연회색80~85타아쉬움 🟣보라86~89타노력 🔴빨강90타 이상
스코어 외에도 티샷·퍼트·로스트볼·비용 등 모든 항목에 동일 등급 기준 적용.

🛠 기술 스택
항목내용프레임워크Vanilla JS (프레임워크 없음)DBSQLite WASM (sql.js) — 100% 로컬오프라인Service Worker앱 패키지Android WebView APK (Android Studio)스타일진초록 테마 (#217346), CSS 변수아이콘SVG 인라인

📦 설치 방법
방법 1 — 브라우저 (PWA)
bash# 로컬 서버 실행
cd mobile_web
python -m http.server 8000
브라우저에서 http://localhost:8000 접속 후
주소창 우측 설치 버튼 클릭 → 홈 화면에 추가
방법 2 — Android APK 직접 설치
apk/ 폴더의 빌드된 APK를 폰에 전송 후 설치
(개인용 독립 APK → Google Play 배포 없이 직접 설치)
방법 3 — GitHub Pages (PWA)
저장소를 Fork 후 GitHub Pages 활성화
→ https://[username].github.io/[repo]/ 로 접속 및 홈 화면 설치

🔄 PC ↔ 폰 데이터 동기화
폰 앱 → 내보내기(.db) → PC 폴더 복사 → 병합 스크립트 실행
PC DB  → 폰 앱 가져오기(.db) → 자동 병합 (중복 제외)
동일 라운드 판정 기준: 날짜 + CC명 + 부 + 시간
